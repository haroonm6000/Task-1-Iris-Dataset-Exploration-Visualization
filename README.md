📋 Overview
This repository contains the completed tasks for the Data Science & Analytics Internship. The project focuses on core data science skills including data exploration, visualization, and machine learning model building using Python libraries such as pandas, matplotlib, seaborn, and scikit-learn.
 
Completed Tasks
Task 1: Iris Dataset Exploration
•	Objective: Understand data structures and basic visualization.
•	Approach: Utilized seaborn to load the dataset and created scatter plots, histograms, and box plots to analyze variable relationships and distributions.
•	Insight: Identified clear clusters between species based on petal length and width.
Task 2: Credit Risk Prediction
•	Objective: Predict loan applicant defaults.
•	Approach: Handled missing values in the Loan Prediction Dataset and trained a Logistic Regression model. Evaluated performance using a confusion matrix.
•	Insight: Credit history was the most significant predictor of loan approval status.
Task 3: Customer Churn Prediction
•	Objective: Identify bank customers likely to exit.
•	Approach: Cleaned the Churn Modelling Dataset and used Label Encoding for categorical features like Geography and Gender.
•	Model: Random Forest Classifier.
•	Insight: Age and Account Balance were identified as primary drivers for customer churn.
Task 4: Predicting Insurance Claim Amounts
•	Objective: Estimate medical charges based on personal attributes.
•	Approach: Developed a Linear Regression model. Evaluated error using MAE and RMSE metrics.
•	Insight: Smoking status and BMI showed the highest positive correlation with medical insurance costs.
Task 5: Personal Loan Acceptance Prediction
•	Objective: Predict which customers accept personal loan offers.
•	Approach: Analyzed the UCI Bank Marketing Dataset focusing on job types and marital status.
•	Model: Decision Tree Classifier.
•	Insight: Contact duration and previous marketing outcomes significantly influenced acceptance rates.
